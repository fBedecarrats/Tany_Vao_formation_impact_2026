5  Expérimentations par assignation aléatoire

Cliquer ici pour télécharger la présentation

Ce chapitre est conservé comme benchmark conceptuel. Dans le cas des aires protégées à Madagascar, il est clair que le traitement n’a pas été assigné aléatoirement. L’intérêt de cette séquence est donc ailleurs : elle sert à montrer ce que l’on attend d’une comparaison crédible quand traités et contrôles sont comparables dès le départ.

5.1 Pourquoi garder cette séquence

La randomisation fournit un point de référence utile pour la suite du parcours. Elle aide à comprendre :

  • pourquoi l’équilibre initial des groupes est si important ;
  • pourquoi une différence moyenne peut recevoir une interprétation plus directe dans un cadre expérimental ;
  • pourquoi, dans les chapitres suivants, il faudra reconstruire une comparabilité plus crédible.

5.2 Préparer une version compacte des données

On repart de l’objet pré-calculé enrichi avec mapme.biodiversity, puis on garde un petit nombre de covariables utiles pour illustrer l’idée d’équilibre.

Code 
library(sf)
library(tidyverse)
library(lubridate)
library(mapme.biodiversity)

AP_Vahatra <- readRDS("data/AP_Vahatra_mapme.rds")

AP_RCT <- AP_Vahatra %>%
    portfolio_wide() %>%
    rename(
        Altitude_moyenne_m = `elevation_2000-02-01_elevation_mean_m`,
        Pente_moyenne_degres = `slope_2000-02-01_slope_mean_degrees`,
        Temps_trajet_minutes = `traveltime_2015-01-01_5k_110mio_traveltime_mean_minutes`,
        Couvert_2000_ha = `treecover_area_2000-01-01_treecover_ha`,
        Couvert_2014_ha = `treecover_area_2014-01-01_treecover_ha`
    ) %>%
    mutate(
        Groupe = ifelse(year(date_creation) < 2015, "Traitement", "Contrôle"),
        Traitement = ifelse(Groupe == "Traitement", 1, 0),
        Surface_ha = as.numeric(st_area(.)) / 10000,
        Couvert_2000_pct = (Couvert_2000_ha / Surface_ha) * 100,
        taux_deforestation_2000_2014 =
            -((Couvert_2014_ha - Couvert_2000_ha) / Couvert_2000_ha) * 100
    )

5.3 Lire un équilibre initial

Dans un cadre réellement randomisé, on espère que les groupes se ressemblent en moyenne avant traitement. Une manière simple d’illustrer cela ici consiste à comparer quelques moyennes descriptives.

Code 
AP_RCT %>%
    st_drop_geometry() %>%
    group_by(Groupe) %>%
    summarise(
        Surface_ha = mean(Surface_ha, na.rm = TRUE),
        Couvert_2000_pct = mean(Couvert_2000_pct, na.rm = TRUE),
        Altitude_moyenne_m = mean(Altitude_moyenne_m, na.rm = TRUE),
        .groups = "drop"
    )
# A tibble: 2 × 4
  Groupe     Surface_ha Couvert_2000_pct Altitude_moyenne_m
  <chr>           <dbl>            <dbl>              <dbl>
1 Contrôle       66540.             62.1               519.
2 Traitement     63728.             72.0               579.

L’objectif n’est pas d’épuiser tous les diagnostics possibles, mais de faire apparaître l’idée centrale : si les groupes diffèrent déjà au départ, la comparaison brute du résultat final devient difficile à interpréter.

5.4 Lire une différence de résultat

On peut ensuite comparer un indicateur simple de résultat, ici le taux de déforestation entre 2000 et 2014.

Code 
AP_RCT %>%
    st_drop_geometry() %>%
    group_by(Groupe) %>%
    summarise(
        taux_deforestation_2000_2014 =
            mean(taux_deforestation_2000_2014, na.rm = TRUE),
        .groups = "drop"
    )
# A tibble: 2 × 2
  Groupe     taux_deforestation_2000_2014
  <chr>                             <dbl>
1 Contrôle                           8.76
2 Traitement                         4.56

5.5 Une régression minimale

La régression suivante ne change pas la logique de fond. Elle reformule la même comparaison dans un langage très courant en économétrie.

Code 
modele_rct <- lm(taux_deforestation_2000_2014 ~ Groupe,
                                 data = AP_RCT %>% st_drop_geometry())

broom::tidy(modele_rct)
# A tibble: 2 × 5
  term             estimate std.error statistic  p.value
  <chr>               <dbl>     <dbl>     <dbl>    <dbl>
1 (Intercept)          8.76      1.20      7.32 8.09e-11
2 GroupeTraitement    -4.20      1.78     -2.36 2.01e- 2

Le coefficient associé au groupe indique ici la différence moyenne de résultat entre traitement et contrôle. Dans un vrai RCT, cette lecture serait beaucoup plus défendable. Dans notre cas, elle sert surtout à montrer ce qu’il manque lorsque la comparabilité initiale n’est pas garantie.

5.6 Ce qu’il faut retenir

La randomisation n’est pas ici une stratégie empirique crédible pour le cas étudié. En revanche, elle joue un rôle pédagogique décisif : elle permet de comprendre pourquoi les chapitres suivants vont se concentrer sur la construction d’un groupe de comparaison plus plausible.
